Introducción

Este projecto se trata de una webapp para crear una lista de tareas sencilla. La API está creada con node.js, Express y Typescript.

La App utiliza su propia API sencilla para las tareas.

Para la parte del Frontend, se usará varios bibliotecas y/o frameworks basados en Javscript. Estos serán:

  • React
  • Vue 3
  • Angular
  • Solid
  • Svelte
